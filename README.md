# Student Mangament System:
This is a Student Management System web application built using Flask (Python), HTML, CSS, and JavaScript.
It helps to manage student data like adding students, tracking attendance, and viewing reports.

Main Features:
The system includes multiple modules:

Student Management:
1.Add, edit, delete students
2.Search students easily

Dashboard:
1.Shows total students, average age, total courses
2.Also displays insights like popular course and latest student

Attendance System:
1.Mark attendance (Present/Absent)
2.View attendance history
3.Generate reports and percentages

Settings Page:
1.Update profile
2.Change password
3.Switch theme (Light/Dark mode)
4.Logout

Authentication System
Login and Register pages for users

💾 Backend
The backend is built using Flask, where routes handle different pages and database operations.

Example:
/ → Students page
/attendance → Attendance system
/settings → Settings page

Database:
It uses a database (like SQLite) to store:
Student details
Attendance records
User login data

Frontend:
“Frontend is built using HTML, CSS, and JavaScript with a clean UI design.
Responsive layout
Navbar for navigation
Cards for dashboard
Dark/light theme support

 Workflow:
 “User logs in → accesses dashboard → manages students → tracks attendance → views reports.

Conclusion:
This project demonstrates full-stack development with frontend, backend, and database integration in a real-world use case.
